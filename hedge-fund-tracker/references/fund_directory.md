# Fund Directory — HedgeFollow URL Slugs

Common funds the user is likely to ask about, with their HedgeFollow URL slug.
URLs follow the pattern `https://hedgefollow.com/funds/<slug>` where `<slug>`
is the fund name with spaces replaced by `+`.

## Notes

- The `+` is URL-encoded for space.
- `%26` is URL-encoded `&`.
- If a fund isn't here, search HedgeFollow directly or try the obvious slug
  pattern. The site has a search box at the top of every page.
- If a slug fails (404), the fund may have a different official filing name
  (e.g. "Berkshire Hathaway" not "Berkshire Hathaway Inc").
